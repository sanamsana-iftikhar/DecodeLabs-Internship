# Project 2: Pizza Point – Responsive Website

## 📖 Project Overview

Pizza Point is a modern and fully responsive restaurant website developed using **HTML5** and **CSS3**. The website is designed to provide an attractive and user-friendly experience on desktops, tablets, and mobile devices.

## ✨ Features

* Fully responsive design
* CSS Media Queries
* Mobile-friendly navigation
* Modern and clean user interface
* Hero section
* About section
* Pizza menu
* Special offers
* Gallery
* Contact section
* Smooth layout with proper spacing and alignment

## 🛠️ Technologies Used

* HTML5
* CSS3
* CSS Flexbox
* CSS Media Queries

## 🎯 Project Objective

This project was created as part of the **DecodeLabs Frontend Development Internship – Project 2 (Responsive Web Layout)**. The objective was to build a responsive webpage that adapts smoothly to different screen sizes while following modern frontend development practices.

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open the `index.html` file in any modern web browser.

**Or**

* Open the project in Visual Studio Code.
* Install the **Live Server** extension (if not already installed).
* Right-click on `index.html` and select **"Open with Live Server"**.

## 📱 Responsiveness

The website is optimized for:

* Mobile Devices
* Tablets
* Laptops
* Desktop Screens
