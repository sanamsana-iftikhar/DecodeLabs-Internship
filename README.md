# 📚 Book Nest

Book Nest is a static website developed as **Project 1** for the **DecodeLabs Frontend Development Internship**.

## 📖 Project Overview
This project was built following the Project 1 guidelines, focusing on creating a clean, semantic, and well-structured static webpage using HTML and CSS.

## 🎯 Objectives
- Create a static webpage using HTML and CSS
- Maintain proper HTML structure
- Use semantic elements
- Design a clean and readable layout
- Apply CSS styling for an attractive interface

## ✨ Features
- Responsive navigation bar
- Hero section
- Book collection section
- About section
- Contact section
- Footer
- Clean and user-friendly design

## 🛠️ Technologies Used
- HTML5
- CSS3
- VS Studio
 ## ▶️ How to Run
1. Download or clone this repository.
2. Make sure both `index.html` and `style.css` are in the same folder.
3. Open `index.html` in any modern web browser (Google Chrome, Microsoft Edge, or Firefox).

The website will load automatically.
